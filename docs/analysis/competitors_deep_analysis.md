# Углублённый анализ конкурентов

## Цель
Выявить сильные и слабые стороны существующих сервисов поиска и сравнения товаров, чтобы принять обоснованные технические и функциональные решения для «Выгодного Взгляда».

## Конкуренты для анализа
1. **YoloPrice** (мобильное приложение, мониторинг цен с интеграцией через OAuth)
2. **Cheaper.ru** (мобильное приложение, честный парсинг)
3. **Price.ru** (веб-сервис, CPA-модель, каталог от партнёров)
4. **Яндекс Маркет** (веб + приложение, поиск с ИИ-чатом)

## Детальный анализ по каждому конкуренту

### 1. YoloPrice
**Сильные стороны:**
- Интеграция через OAuth (безопасно, не нужно передавать пароль).
- Push-уведомления о снижении цены.
- Персонализация через корзину и избранное пользователя.

**Слабые стороны:**
- Нет веб-версии (только мобильное приложение).
- Непонятно, как монетизируется (риск закрытия).
- Требует авторизации на маркетплейсах (барьер для пользователя).

**Что взять в наш проект:** идею мониторинга цен и уведомлений (отложено на спринт 2). Механизм OAuth – сложно на MVP, но можно рассмотреть позже.

### 2. Cheaper.ru
**Сильные стороны:**
- Простота: честный парсинг, без лишних сущностей.
- Не требует регистрации.

**Слабые стороны:**
- Постоянные блокировки (CAPTCHA, бан IP).
- Низкая релевантность (дубликаты).
- Отсутствие персонализации.

**Что взять в наш проект:** использовать их опыт – необходимо реализовать защиту от блокировок (ротация User-Agent, задержки, прокси). Избежать их ошибок с дубликатами (добавить группировку товаров).

### 3. Price.ru
**Сильные стороны:**
- Высокая надёжность данных (фиды от партнёров).
- Быстрый поиск по своей БД.
- Коммерчески устойчивая модель.

**Слабые стороны:**
- Закрытый каталог (не находит то, чего нет у партнёров).
- Нет персонализации и сравнения в реальном времени.

**Что взять в наш проект:** в будущем, при масштабировании, можно рассмотреть подключение партнёрских фидов как дополнительный источник данных. Но для MVP – парсинг.

### 4. Яндекс Маркет
**Сильные стороны:**
- Огромная база товаров.
- ИИ-чат для помощи в выборе.
- Высокая релевантность поиска.

**Слабые стороны:**
- Сложный интерфейс (иногда перегружен).
- Не всегда понятно, как формируется цена (со скидками, доставкой).

**Что взять в наш проект:** концепцию интеллектуальной помощи пользователю (наш «индекс выгоды» – упрощённый аналог ИИ-рекомендаций). Простой и понятный интерфейс – наше преимущество.

## Выводы и рекомендации для нашего проекта

| Аспект | Рекомендация |
|--------|---------------|
| Сбор данных | Начать с парсинга Ozon и WB (как Cheaper.ru), но добавить защиту от блокировок и кеширование. |
| Персонализация | Сделать настройку весов (как у YoloPrice, но проще) – это наше ключевое отличие. |
| Точность | Реализовать группировку одинаковых товаров (через сравнение названий и артикулов). |
| Интерфейс | Ориентироваться на минимализм Яндекс.Маркета, но без перегруза. |
| Мониторинг | Отложить на спринт 2 (как у YoloPrice) – функция подписки на изменение цены. |
| Блокировки | Использовать ротацию User-Agent, задержки, прокси-пулы (на основе опыта Cheaper.ru). |

## Заключение
Ни один из существующих сервисов не сочетает в себе одновременно: честный кросс-маркетплейсный поиск, персонализацию через веса пользователя и простой интерфейс. Наш проект «Выгодный Взгляд» имеет потенциал закрыть эту нишу в рамках учебной задачи.
