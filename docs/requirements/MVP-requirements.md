# MVP Requirements v1.0

## Обязательный функционал (должно работать к концу спринта 2)

1. **Поиск товара** по точному названию или артикулу.
   - Вход: текстовая строка (например, "Sony WH-1000XM5").
   - Выход: таблица предложений с Ozon и Wildberries.

2. **Сбор данных** с двух площадок:
   - Цена (в рублях)
   - Срок доставки (в днях, средний по Москве)
   - Рейтинг продавца (0–5)
   - Ссылка на товар

3. **Ручная настройка весов** пользователем:
   - Три слайдера: Цена (0..1), Доставка (0..1), Рейтинг (0..1)
   - Сумма весов не обязательно равна 1, нормализация внутри алгоритма.

4. **Расчёт индекса выгоды**:
   - Для каждого предложения: `benefit = (1/price)*w_price + (1/delivery_days)*w_delivery + rating*w_rating`
   - Сортировка по убыванию benefit.

5. **Отображение результатов** в виде карточек/таблицы с кнопкой "Перейти на маркетплейс".

## НЕ входит в MVP (на потом)
- Авторизация пользователей (профиль без сохранения между сессиями)
- Отслеживание цены в динамике
- Уведомления
- Фото-поиск
- Более 2 маркетплейсов
