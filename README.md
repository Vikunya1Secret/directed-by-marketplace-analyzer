# Выгодный Взгляд (Directed By)

Интеллектуальный веб-сервис поиска и сравнения товаров на маркетплейсах.

**Команда:** Directed By  
**Куратор:** Харисов Азамат Робертович  
**Учебный проект:** УрФУ, Институт радиоэлектроники и информационных технологий – РТФ

## Цель проекта
Создать централизованную интеллектуальную систему, которая анализирует предложения на Ozon, Wildberries и других площадках, учитывает персональные приоритеты пользователя (цена, скорость доставки, рейтинг продавца) и рекомендует оптимальный вариант покупки.

## Технологический стек (планируемый)
- **Парсинг:** Python (BeautifulSoup, requests, aiohttp)
- **Бэкенд:** FastAPI (Python)
- **Фронтенд:** React + Bootstrap
- **База данных:** SQLite (MVP) → PostgreSQL
- **Деплой:** Docker + GitHub Actions

## Структура репозитория
- `/docs` – документация, требования, отчёты
- `/src/parser` – модули парсинга Ozon и Wildberries
- `/src/backend` – API сервер
- `/src/frontend` – пользовательский интерфейс
- `/tests` – автотесты

## Как запустить (будет дополняться)
1. Клонировать репозиторий
2. Установить зависимости для парсера: `pip install -r src/parser/requirements.txt`
3. Запустить парсер тестово: `python src/parser/ozon_parser.py --query "наушники"`
