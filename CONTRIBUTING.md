# Правила работы в репозитории

## Ветки
- `main` – только стабильные релизы (защищена, только через pull request)
- `dev` – интеграционная ветка для разработки
- `feature/*` – для новых функций (например, `feature/ozon-parser`)
- `fix/*` – для исправлений
